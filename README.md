# CELSSICA - Cumplimiento Venezuela para Odoo 17

Módulo de localización venezolana desarrollado por CELSSICA.

## Características

- Plan de cuentas básico Venezuela
- Impuestos: IVA 16%, Retenciones IVA 75%/100%
- ISLR: Conceptos y retenciones
- Unidad Tributaria (UT) configurable
- Campos RIF y número de control en facturas

## Instalación

1. Copiar a addons/
2. Actualizar lista de aplicaciones
3. Instalar "CELSSICA - Cumplimiento Venezuela"
